请求头

我们在这里介绍一些在我们的请求报文中会遇到的请求头函数

Host

代表着我们要请求的目标地址

比如说：在浏览器中，当用户在浏览器中访问 https://example.com 时，浏览器会自动设置 Host 头部字段为 example.com。

User-Agent

User-Agent 头部字段包含客户端的识别信息，通常包括以下几个部分：

  • 浏览器名称和版本
  • 操作系统名称和版本
  • 其他客户端信息

X-Forwarded-For

X-Forwarded-For 头部字段包含一个逗号分隔的 IP 地址列表，表示客户端的原始 IP 地址以及请求经过的所有代理服务器的 IP 地址。列表中的第一个 IP 地址通常是客户端的真实 IP 地址，后续的 IP 地址是请求经过的各个代理服务器的 IP 地址。

这个是用来识别客户端的真实ip地址，但是这个是可以伪造的

Referer

用于指示当前请求的来源页面。它提供了当前页面的上一个页面的 URL，即用户在访问当前页面之前访问了哪个页面。Referer 头部字段对于跟踪用户的浏览路径和进行网站分析非常有用。

例如：

当我从https://www.example.com/page1这个网站中去访问其他的网站，那么

Referer为https://www.example.com/page1

表示当前请求是从 https://www.example.com/page1 页面发起的。
